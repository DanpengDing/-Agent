# 知识库 9015

## 标题
雷电预警信号与户外防雷避险

## 问题描述
整理中国气象局公开的雷电预警标准和户外防雷要点，回答雷雨天气在室内、室外和空旷地带如何避险。

## 分类
主类别: 极端天气应对
子类别: 雷电避险

## 关键词
雷电预警, 防雷, 雷雨, 户外避险, 树下避雨, 金属物, 切断电源

创建时间:2026-04-09|版本:1.0|来源类型:气象部门公开指南

## 解决方案
**来源说明：**  
本文依据中国气象局公开的《雷电预警信号标准》和地方应急部门公开防雷要点整理。

**1. 雷电预警分几级？**

雷电预警分三级：

1. 黄色。  
2. 橙色。  
3. 红色。

**2. 黄色预警时怎么做？**

标准是：  
6 小时内可能发生雷电活动，可能造成雷电灾害事故。

防御重点：

1. 密切关注天气变化。  
2. 尽量避免户外活动。

**3. 橙色预警时怎么做？**

标准是：  
2 小时内发生雷电活动可能性很大，或者已经受雷电影响且可能持续。

防御重点：

1. 人员留在室内并关好门窗。  
2. 户外人员立即躲入有防雷设施的建筑物或者汽车内。  
3. 切断危险电源。  
4. 不要在树下、电杆下、塔吊下避雨。  
5. 在空旷场地不要打伞。  
6. 不要把球杆、农具、羽毛球拍等金属长杆扛在肩上。

**4. 红色预警时怎么做？**

标准是：  
2 小时内发生强烈雷电活动的可能性非常大，或者已发生强烈雷电活动且可能持续。

防御重点：

1. 尽量躲入有防雷设施的建筑物或者汽车内，并关好门窗。  
2. 不要接触天线、水管、铁丝网、金属门窗、建筑物外墙。  
3. 远离电线等带电设备。  
4. 尽量不要使用无防雷装置或者防雷装置不完备的电器。  
5. 持续关注最新预警信息。

**5. 雷雨天在室外最容易犯哪些错误？**

1. 躲在大树下。  
2. 躲在亭子、草棚、电杆下。  
3. 在空旷地举伞行走。  
4. 携带金属长杆物品停留。  
5. 在水边、洼地停留。  
6. 继续游泳、骑车、打球或进行水上活动。

**6. 如果来不及进入建筑物怎么办？**

在空旷地带可尽量降低身体高度，双脚并拢，避免成为突出目标。  
不要平躺，也不要靠近大树、金属设施和接地线。

**7. 这篇文档适合回答哪些问题？**

1. 雷电橙色预警还能不能在外面活动。  
2. 雷雨天为什么不能在树下避雨。  
3. 雷电红色预警时能不能碰金属门窗和水管。  
4. 为什么打伞和拿球杆在空旷地危险。  
5. 雷雨天户外人员第一时间该躲到哪里。
