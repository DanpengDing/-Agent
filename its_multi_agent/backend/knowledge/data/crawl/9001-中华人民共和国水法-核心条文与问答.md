# 知识库 9001

## 标题
中华人民共和国水法：核心条文与问答

## 问题描述
整理中国政府网公开的《中华人民共和国水法》核心内容，回答水资源归属、开发利用、节约保护和取水许可等基础问题。

## 解决方案
**来源说明：**  
本文依据中国政府网公开的《中华人民共和国水法》整理，不是二次转载摘要。

**1. 水法主要管什么？**

《中华人民共和国水法》规定，在中华人民共和国领域内开发、利用、节约、保护、管理水资源以及防治水害，适用本法。  
这里的水资源包括地表水和地下水。

**2. 水资源归谁所有？**

水资源属于国家所有，由国务院代表国家行使所有权。  
这意味着地表水、地下水的开发利用不是“谁打到井谁拥有”，而是在国家制度下统一管理。

**3. 开发利用水资源遵循什么原则？**

根据公开法条，开发、利用、节约、保护水资源和防治水害，应当坚持：

1. 全面规划。  
2. 统筹兼顾。  
3. 标本兼治。  
4. 综合利用。  
5. 讲求效益。  
6. 协调生活、生产经营和生态环境用水。

这也是水利业务问答里最常见的总原则。

**4. 县级以上政府承担什么责任？**

县级以上人民政府应当加强水利基础设施建设，并将其纳入本级国民经济和社会发展计划。  
所以水法不是只约束企业或取水户，也明确要求政府承担公共治理责任。

**5. 国家鼓励什么、要求什么？**

国家鼓励单位和个人依法开发、利用水资源，但同时要求开发利用者依法保护水资源。  
也就是说，“开发”和“保护”不是对立关系，而是同时成立的法定义务。

**6. 用水顺序如何理解？**

水资源开发利用应当优先满足城乡居民生活用水，并统筹农业、工业、生态环境等用水需求。  
当用户问“为什么有些地区会限制工业或高耗水项目”，这个原则就是基础依据之一。

**7. 为什么节水是法律要求？**

水法把节约用水放在重要位置。  
在水资源不足地区，应限制城市规模和高耗水工业、农业、服务业项目的发展；地方在开发利用水资源时，应统筹地表水和地下水，开源与节流结合，节流优先。

**8. 取水为什么不是想取就取？**

国家对水资源依法实行取水许可制度和水资源有偿使用制度。  
这意味着大量取用水通常需要经过许可、计量、监管，不是简单的市场行为。

**9. 水法适合回答哪些问题？**

1. 水资源是不是国家所有。  
2. 地表水和地下水是否都受统一法律约束。  
3. 为什么要实行取水许可。  
4. 为什么缺水地区要限制高耗水项目。  
5. 为什么节约用水是法定义务。  
6. 为什么水利管理既涉及政府也涉及企业和个人。

**10. 相关知识库扩展方向**

基于这部法律，还可以继续拆分出：

1. 取水许可制度专题。  
2. 水资源费与水资源税专题。  
3. 节约用水与计划用水专题。  
4. 水资源保护与水工程保护专题。  
5. 水事纠纷和执法监督专题。
