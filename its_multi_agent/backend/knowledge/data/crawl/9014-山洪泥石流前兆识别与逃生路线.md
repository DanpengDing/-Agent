# 知识库 9014

## 标题
山洪泥石流前兆识别与逃生路线

## 问题描述
整理山洪、泥石流的前兆识别、易受威胁人群和正确逃生路线，回答暴雨山区遇险时如何快速判断和撤离。

## 分类
主类别: 暴雨洪水应对
子类别: 山洪泥石流

## 关键词
山洪, 泥石流, 前兆, 逃生路线, 沟谷, 高地, 垂直逃离, 山区暴雨

## 解决方案
**来源说明：**  
本文依据应急管理部公开的山洪泥石流科普内容整理。

**1. 山洪泥石流一般在什么情况下高发？**

1. 多雨的夏秋季。  
2. 一次降雨高峰期。  
3. 连续降雨之后。  
4. 有大量松散堆积物、沟谷陡峻的山区。  
5. 人类不合理切坡建房、侵占沟道等活动区域。

**2. 哪些人群更容易受威胁？**

1. 在切坡建房且无防护的居民。  
2. 房屋位于陡坎或陡坡脚下的居民。  
3. 住在溪河两边低洼处、河道拐弯处、双河口的人。  
4. 暴雨天在高山、陡坡、溪河两边活动的人。  
5. 山洪泥石流发生时还试图抢救财物或匆忙涉河过桥的人。

**3. 山洪泥石流来临前有哪些前兆？**

典型前兆有两类：

1. 看：正常流水突然断流，或者洪水突然变大并夹带柴草、树木，说明上游可能已形成泥石流。  
2. 听：深谷、沟内传来类似火车轰鸣声、闷雷声，或者伴有轻微震动，说明危险正在形成。

**4. 发现前兆后正确的逃生方向是什么？**

1. 立即向沟谷两侧山坡的高处跑。  
2. 逃生方向要与泥石流前进方向垂直。  
3. 绝不能沿沟谷向上游或下游跑。  
4. 不要在沟底、河谷、低洼谷底停留。  
5. 不要横渡泥石流。

**5. 为什么不能回去拿东西？**

泥石流和山洪突发性强、速度快，回头取物极易错过最佳逃生时间。  
已撤到危险区外后，绝不能返回收拾物品或锁门。

**6. 山区旅游遇到暴雨该怎么办？**

1. 暴雨天或连续阴雨后不要贸然进入山区沟谷。  
2. 野外扎营应选高地，避开沟底和堆积物坡脚。  
3. 一旦遇大雨，要迅速撤到高处。  
4. 雨停后也不要立刻回沟谷，应等待一段时间。

**7. 这篇文档适合回答哪些问题？**

1. 山洪和泥石流来前怎么看征兆。  
2. 为什么不能顺着沟跑。  
3. 暴雨后山区旅游为什么危险。  
4. 遇到泥石流时能不能横穿。  
5. 哪些地形和房屋位置最危险。
