# 知识库 9002

## 标题
地下水管理条例：超采治理与污染防治

## 问题描述
整理《地下水管理条例》中的核心制度，覆盖地下水调查规划、总量控制、超采区治理、污染防治和监督管理。

## 分类
主类别: 水资源管理
子类别: 地下水管理

## 关键词
地下水管理条例, 地下水超采, 地下水污染防治, 禁采区, 限采区, 地下水水位控制, 采补平衡, 地下水监测

## 解决方案
**来源说明：**  
本文依据公开的《地下水管理条例》正文整理。

**1. 这部条例管什么？**

《地下水管理条例》适用于地下水调查与规划、节约与保护、超采治理、污染防治、监督管理等活动。  
如果用户问“地下水治理归不归水利管”，这部条例是直接依据。

**2. 地下水管理的原则是什么？**

条例明确，地下水管理坚持：

1. 统筹规划。  
2. 节水优先。  
3. 高效利用。  
4. 系统治理。

这说明地下水问题不是只靠打一口井、封一口井解决，而是要把资源、生态、污染、供水结构一起考虑。

**3. 谁负责地下水管理？**

1. 国务院水行政主管部门负责全国地下水统一监督管理。  
2. 国务院生态环境主管部门负责全国地下水污染防治监督管理。  
3. 国务院自然资源等主管部门按职责做好调查、监测等工作。  
4. 地方政府对本行政区域内地下水管理负责。

**4. 地下水怎么控总量？**

条例要求实行地下水取水总量控制制度。  
国家会制定并下达地下水取水总量控制指标；省级再细化到本行政区域，并同时制定地下水水位控制指标。

这意味着地下水管理不是模糊口号，而是有数量指标和水位指标两套约束。

**5. 哪些取水许可申请不予批准？**

条例列举了不予批准的典型情形，包括：

1. 不符合地下水取水总量控制、水位控制要求。  
2. 不符合限制开采区取用水规定。  
3. 不符合行业用水定额和节水规定。  
4. 不符合强制性国家标准。  
5. 在水资源紧缺或者生态脆弱地区新建、改建、扩建高耗水项目。  
6. 违反法律法规规定开垦种植而取用地下水。

**6. 什么是禁采区和限采区？**

条例要求，各省区市要组织划定地下水禁止开采区、限制开采区。

应划为禁止开采区的典型情形包括：

1. 已发生严重地面沉降、地裂缝、海水入侵、植被退化等问题的区域。  
2. 地下水超采区内，公共供水管网覆盖或者替代水源已经能够满足供水需求的区域。

应划为限制开采区的典型情形包括：

1. 地下水开采量接近可开采量的区域。  
2. 开采地下水可能引发地质灾害或者生态损害的区域。

**7. 超采区怎么治理？**

条例要求编制地下水超采综合治理方案，明确治理目标、治理措施和保障措施。  
治理方向包括：

1. 加强节水型社会建设。  
2. 调整种植结构。  
3. 推广节水农业。  
4. 加强工业节水。  
5. 实施河湖地下水回补。  
6. 逐步实现地下水采补平衡。

**8. 地下水污染防治禁止哪些行为？**

条例明确禁止利用渗井、渗坑、裂隙、溶洞以及私设暗管等逃避监管方式排放水污染物；  
也禁止利用无防渗漏措施的沟渠、坑塘等输送或者贮存有毒污染物废水、含病原体污水和其他废弃物。

对化工企业、工业集聚区、矿山开采区、尾矿库、危险废物处置场、垃圾填埋场等，还要求采取防渗漏措施，并建设地下水水质监测井。

**9. 这部条例最适合回答哪些问题？**

1. 谁负责地下水监管。  
2. 为什么地下水要做总量控制和水位控制。  
3. 什么情况下不能再批地下水取水。  
4. 什么是地下水禁采区、限采区。  
5. 地下水超采和地下水污染分别怎么治理。
