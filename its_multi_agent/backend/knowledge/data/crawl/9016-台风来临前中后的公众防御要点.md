# 知识库 9016

## 标题
台风来临前中后的公众防御要点

## 问题描述
整理中国气象局公开的台风预警和公众防御原则，回答台风来临前、中、后应如何准备和应对。

## 分类
主类别: 极端天气应对
子类别: 台风防御

## 关键词
台风, 台风预警, 公众防御, 灾前准备, 居家防台, 高空作业, 回港避风

## 解决方案
**来源说明：**  
本文依据中国气象局公开的台风预警信号及国家标准《公众气象灾害防御行为指南 台风》介绍内容整理。

**1. 台风公众防御的总原则是什么？**

中国气象局公开介绍的国家标准提出三条总体原则：

1. 提前准备、防患未然。  
2. 分级防御、科学应对。  
3. 以人为本、生命至上。

**2. 台风蓝色预警时重点做什么？**

公开防御指南显示，蓝色预警阶段重点包括：

1. 做好防台风准备。  
2. 停止露天集体活动和高空等户外危险作业。  
3. 水上作业和过往船舶积极采取回港避风或者绕行措施。  
4. 加固门窗、围板、棚架、广告牌等易被吹动设施。  
5. 切断危险的室外电源。

**3. 台风来临前家庭要准备什么？**

1. 提前了解权威预警信息和可能影响时段。  
2. 制定家庭应急计划。  
3. 备好饮用水、食品、照明、充电设备、常用药品等。  
4. 收好阳台、窗外、楼顶容易被吹落的物品。  
5. 尽量避免非必要外出。

**4. 台风影响期间重点注意什么？**

1. 尽量留在安全室内。  
2. 远离门窗、广告牌、临时搭建物、树木和高空坠物风险区。  
3. 不前往海边、河边、低洼地和山洪地质灾害风险区。  
4. 严格根据预警级别加大防御力度。  
5. 必要时服从当地政府和有关部门统一转移安排。

**5. 台风过后为什么不能马上放松？**

台风过后仍可能出现：

1. 持续强降雨。  
2. 城市内涝。  
3. 山洪、滑坡、泥石流。  
4. 电力设施损坏和树木倒伏。  
5. 海边、河边次生风险。

因此灾后也应确认环境安全后再外出和恢复生产生活。

**6. 这篇文档适合回答哪些问题？**

1. 台风来之前家里要准备什么。  
2. 蓝色台风预警就要停止哪些高风险活动。  
3. 为什么台风天不能去海边和低洼地。  
4. 为什么台风走了仍要防强降雨和次生灾害。  
5. 台风防御为什么强调分级响应。
