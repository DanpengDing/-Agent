# 知识库 9004

## 标题
全面推行河长制与湖长制：职责任务与治理重点

## 问题描述
整理中国政府网公开的河长制、湖长制文件核心内容，回答河湖长体系、主要职责和湖泊治理重点。

主类别: 河湖管理
子类别: 制度建设

## 关键词

## 解决方案
**来源说明：**  
本文依据中国政府网公开的《关于全面推行河长制的意见》和《关于在湖泊实施湖长制的指导意见》整理。

**1. 为什么要推行河长制？**

中国政府网公开文件指出，河湖管理保护涉及上下游、左右岸、不同行政区域和行业，是复杂的系统工程。  
全面推行河长制，是为了解决复杂水问题、维护河湖健康生命、完善水治理体系。

**2. 河长制的主要任务是什么？**

公开意见明确，河长制以保护水资源、防治水污染、改善水环境、修复水生态为主要任务。  
这四项任务基本覆盖了大多数河湖治理类问答。

**3. 河长体系怎么建？**

文件要求建立以党政领导负责制为核心的责任体系，形成一级抓一级、层层抓落实的工作格局。  
实践中通常形成省、市、县、乡多级河长体系。

**4. 湖长制为什么要单独强调？**

湖泊与一般河道相比有明显特殊性：

1. 河湖关系复杂，往往有多条河流汇入。  
2. 湖区边界和责任划分更难。  
3. 围垦、养殖、旅游开发等活动更容易导致无序开发。  
4. 湖泊水体更新慢，污染后修复难度大。  
5. 对区域生态平衡和生物多样性影响更大。

**5. 湖长制的核心职责是什么？**

公开指导意见提出，湖泊最高层级湖长是第一责任人，对湖泊管理保护负总责；  
其他各级湖长对本辖区内管理保护负直接责任。  
核心工作包括统筹湖泊与入湖河流管理、组织制定“一湖一策”、协调解决重大问题、整治突出违法行为。

**6. 湖泊治理重点有哪些？**

根据公开文件，重点包括：

1. 严格湖泊水域空间管控。  
2. 强化湖泊岸线管理保护。  
3. 加强湖泊水资源保护和水污染防治。  
4. 开展湖泊生态治理与修复。  
5. 健全执法监管机制。  
6. 动态监测湖泊变化情况。

**7. 哪些涉湖行为会被重点整治？**

文件明确提到要坚决清理整治：

1. 围垦湖泊。  
2. 侵占水域。  
3. 非法排污。  
4. 违法养殖。  
5. 非法采砂。  
6. 设障、捕捞、取用水等违法违规活动。

**8. 这类制度适合回答哪些问题？**

1. 河长制和湖长制是什么。  
2. 河长、湖长负责什么。  
3. 什么是一河一策、一湖一策。  
4. 湖泊治理为什么比河道更复杂。  
5. 河湖治理为什么强调跨区域协同。
