# 知识库 9012

## 标题
城市内涝避险与车辆涉水逃生

## 问题描述
整理城市内涝发生时的室内、室外、步行和驾车避险措施，回答积水路段、地下空间和车辆被困时如何自救。

子类别: 城市内涝

## 关键词

## 解决方案
**来源说明：**  
本文依据中国消防、应急管理部门发布的城市内涝避险内容整理。

**1. 城市内涝最容易发生在哪些地方？**

高风险地点主要包括：

1. 城区低洼地区。  
2. 下凹式立交桥。  
3. 地下轨道交通、地下商场、地下车库、地下室等地下空间。  
4. 危旧房屋。  
5. 在建工地。

**2. 暴雨来临前应该提前做什么？**

1. 关注天气预报和内涝风险预警。  
2. 尽量避免前往易涝点和地下空间。  
3. 地势低洼住户可提前用门板、沙袋等挡水。  
4. 家中和车内可常备手电、救生衣、救生圈、安全锤等用品。

**3. 如果室内开始进水怎么办？**

1. 及时切断屋内电源和气源。  
2. 不要继续停留在屋内低处。  
3. 迅速撤离到附近高地、楼上或楼顶。  
4. 不要回头抢拿财物而延误撤离。

**4. 如果在室外步行遇到积水怎么办？**

1. 不要贸然走入积水深处。  
2. 远离水面漩涡，防止跌入缺失井盖和地坑。  
3. 尽量贴近建筑物、高处行走。  
4. 避开灯杆、电线杆、变压器、铁栏杆、树木等可能导电的物体。  
5. 最好不要光脚蹚水，防止感染和割伤。  
6. 必须涉水时，可用长杆或结实长柄雨伞探路。

**5. 为什么积水里要格外注意触电？**

暴雨天气下，电线杆、路灯、空调外机、断落电线附近都可能漏电。  
如果发现高压线塔倾斜、电线垂落到积水中，要立即远离并报警，不要自行处置。

**6. 驾车遇到积水路段应该怎么办？**

1. 先观察水深，不明深浅不要强行通过。  
2. 下凹式立交桥、低洼地带尽量绕行。  
3. 不要把车停在低洼处避雨。  
4. 如果积水已经漫过车轴且仍在上涨，应及时弃车逃生。  
5. 车辆涉水熄火后，切勿再次启动发动机。

**7. 被困车里时怎么逃生？**

1. 保持冷静，优先尝试开门开窗撤离。  
2. 如无法正常打开车门车窗，使用安全锤等工具敲击侧窗边角。  
3. 如积水持续上涨，应尽快离车，不要把车当成避雨点。  
4. 撤离后迅速前往高处，不要停留在车旁。

**8. 这篇文档适合回答哪些问题？**

1. 城市内涝时为什么不能去地下车库和地下商场。  
2. 积水中为什么要避开漩涡和井盖。  
3. 车辆进水熄火后能不能二次启动。  
4. 水深到什么程度应弃车。  
5. 室内进水后第一步为什么是断电断气。
