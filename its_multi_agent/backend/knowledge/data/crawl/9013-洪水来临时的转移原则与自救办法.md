## 标题
洪水来临时的转移原则与自救办法

## 问题描述
整理洪水来临时的转移原则、临时避险位置和洪水过后注意事项，回答被洪水围困时如何自救。

## 解决方案
**来源说明：**  
本文依据应急管理部和水利系统公开的洪水自救知识整理。

**1. 洪水来前最重要的原则是什么？**

时间充裕时，应按照预定路线有组织地向山坡、高地、坚固楼房上层等安全地点转移。  
洪水逃生最怕犹豫不决，出现“人未走，水先到”。

**2. 已经被洪水包围时怎么办？**

1. 优先利用船只、木排、门板、木床等漂浮物做临时转移工具。  
2. 如果来不及转移，立即爬上屋顶、高楼、高墙或大树等暂时避险。  
3. 不要单独游泳转移。  
4. 尽快与当地政府或救援部门联系，报告自己的位置和险情。

**3. 在山区遭遇洪水要特别注意什么？**

1. 连续大雨后，山区容易突发山洪。  
2. 不要强行涉水过河、过桥。  
3. 同时警惕滑坡、滚石、泥石流。  
4. 若发现河水陡涨，应立刻向高处撤离，不要沿河道停留。

**4. 洪水期间为什么要远离电力设施？**

如发现高压线铁塔倾倒、电线低垂、断线入水等情况，要立刻远离，不可接近或触摸，防止触电。  
这是洪水避险中的高频致命风险点。

**5. 洪水过后还要注意什么？**

1. 做好卫生防疫，避免饮用受污染生水。  
2. 必要时服用防流行病药物，遵从当地防疫提醒。  
3. 食物注意加热和清洁，防止肠道传染病。  
4. 返回住所前先确认房屋结构、电路、燃气是否安全。

**6. 这篇文档适合回答哪些问题？**

1. 洪水围困时应该往哪里撤。  
2. 为什么不能游泳逃生。  
3. 洪水中能不能靠近电线杆。  
4. 山区洪水和普通城市积水有什么不同。  
5. 洪水过后为什么还要防病防疫。
