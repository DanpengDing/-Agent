# 知识库 9011

## 标题
暴雨预警信号与对应防御措施

## 问题描述
整理中国气象局公开的暴雨预警信号标准和分级防御指南，回答不同级别暴雨预警下应该采取什么措施。

## 分类
主类别: 暴雨洪水应对
子类别: 预警响应

## 关键词
暴雨预警, 蓝色预警, 黄色预警, 橙色预警, 红色预警, 暴雨防御, 强降雨

## 元信息
创建时间:2026-04-09|版本:1.0|来源类型:气象部门公开指南

## 解决方案
**来源说明：**  
本文依据中国气象局公开的《暴雨预警信号标准》整理。

**1. 暴雨预警分几级？**

暴雨预警信号分四级：

1. 蓝色。  
2. 黄色。  
3. 橙色。  
4. 红色。

风险从低到高递增。

**2. 蓝色预警意味着什么？**

标准是：  
12 小时内降雨量将达 50 毫米以上，或者已达 50 毫米以上且降雨可能持续。

对应防御措施：

1. 做好防暴雨准备工作。  
2. 学校、幼儿园采取适当措施保障学生安全。  
3. 驾驶人员注意道路积水和交通阻塞。  
4. 检查城市、农田、鱼塘排水系统，做好排涝准备。

**3. 黄色预警意味着什么？**

标准是：  
6 小时内降雨量将达 50 毫米以上，或者已达 50 毫米以上且降雨可能持续。

对应防御措施：

1. 相关部门按职责做好防暴雨工作。  
2. 交通管理部门可对强降雨路段采取交通管制。  
3. 切断低洼地带有危险的室外电源。  
4. 暂停空旷地方户外作业。  
5. 转移危险地带人员和危房居民到安全场所。  
6. 采取必要排涝措施。

**4. 橙色预警意味着什么？**

标准是：  
3 小时内降雨量将达 50 毫米以上，或者已达 50 毫米以上且降雨可能持续。

对应防御措施：

1. 做好防暴雨应急工作。  
2. 切断有危险的室外电源。  
3. 暂停户外作业。  
4. 危险地带单位停课、停业，并保护在校学生和在岗人员安全。  
5. 做好城市和农田排涝。  
6. 注意防范山洪、滑坡、泥石流等次生灾害。

**5. 红色预警意味着什么？**

标准是：  
3 小时内降雨量将达 100 毫米以上，或者已达 100 毫米以上且降雨可能持续。

对应防御措施：

1. 做好防暴雨应急和抢险工作。  
2. 停止集会、停课、停业，特殊行业除外。  
3. 重点做好山洪、滑坡、泥石流防御和抢险。

**6. 公众在暴雨升级时怎么理解行动重点？**

可以简单理解为：

1. 蓝色预警重在准备。  
2. 黄色预警重在减少外出和排查隐患。  
3. 橙色预警重在停止危险活动和及时转移。  
4. 红色预警重在非必要不外出、严格避险和服从统一安排。

**7. 这篇文档适合回答哪些问题？**

4. 红色预警时重点防什么次生灾害。
